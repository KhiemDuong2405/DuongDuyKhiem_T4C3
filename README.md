## 🛠️ Cài đặt

Để cài đặt và chạy dự án trên máy của bạn, làm theo các bước dưới đây:

```bash
# Clone repo
git clone https://github.com/your-username/your-repo.git

# Cài đặt các thư viện cần thiết
npm install

# Chạy ứng dụng
npm start
